# CS3D
Basic 3D Framework (On its way to Advanced 3D Framework) Runs in OpenGL 3.3 On Windows, Linux and Mac.

How to build
---------

Windows, Linux, Mac:

	mkdir build
	cd build
	cmake -Wno-dev ..
	make
	./CS3D_Project

To launch the editor instead of the project:

	./CS3D_Editor
